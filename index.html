Thanks for purchasing from the CyberSphere Innovations product the BrandRise Solutions webite template. This is the documentation guide to help you, how to implement and customize the site for your choice.

Hero Aria
Customizing the Hero Aria

To change the hero aria header title go to the main_template folder (Where the website full code has been stored and provided), go to the index.html file and on the hero aria section change the hero aria header on the h1 tag and you can change the sub title on the h4 tag and customize these as your own.


To add or change the background image of the hero aria go to the css folder. On the css folder you will find the style.main.css file. Open the file, and on the hero aria section you will find the background and on the background there is a url. On the url you have to specify the img path. To set the image you have to just set the image path.

NOTE : It is recommanded to store all the images on the img folder.
